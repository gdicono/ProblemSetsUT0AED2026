# ProblemSetUT0AED2026

## Objetivo del repositorio:

Reforzar conocimiento sobre Java a través de la resolución de ejercicios prácticos.

Areas de enfoque:
- Fundamentos y flujos de control. Uso de variables, tipos primitivos y de referencia, estructuras de datos, y repeticion.
- Programación Orientada a Objetos. Clases, objetos, encapsulamiento, herencia, polimorfismo, y manejo de excepciones.
- Manipulacion de texto y archivos. Lectura y escritura de archivos, manejo de cadenas, expresiones regulares, y procesamiento de texto.
- Entrada/Salida y colecciones. Manejo de archivos, lectura desde consola con Scanner e implementación de arreglos y colecciones de Java.
- Calidad de software. Uso de JUnit para pruebas unitarias, pruebas parametrizadas, manejo de excepciones en tests, y validación de rendimiento con timeout.

## Estructura del proyecto:
- src/main/java: Código fuente de la aplicación.
- src/test/java: Código fuente de las pruebas unitarias.
- pom.xml: Configuración de dependencias y construcción del proyecto.
